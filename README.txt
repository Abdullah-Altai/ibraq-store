متجر إبراق — نسخة عربي/إنجليزي مع لوحة إدارة

طريقة التشغيل:
1) فك الضغط.
2) افتح index.html لعرض المتجر.
3) افتح admin.html للوحة الإدارة.
4) بيانات الدخول الافتراضية:
   المستخدم: admin
   كلمة المرور: admin123

ما تستطيع التحكم به:
- المنتجات باللغتين، السعر، المخزون، الترتيب، الصور المتعددة، الإظهار والإخفاء.
- الأقسام باللغتين.
- الإعلانات المتحركة وربط كل إعلان بمنتج.
- اسم المتجر، الوصف، من نحن، الألوان، العملة، الشعار، رقم واتساب.
- اسم المستخدم وكلمة مرور الإدارة.
- تصدير واستيراد نسخة احتياطية JSON.

ملاحظة مهمة:
هذه النسخة تحفظ البيانات في LocalStorage داخل نفس المتصفح والجهاز.
حتى تكون البيانات مشتركة بين الهاتف والكمبيوتر وتعمل أونلاين لكل المستخدمين،
يجب ربط المشروع لاحقاً بقاعدة بيانات مثل Supabase ورفعه على استضافة.

تحديث الإصلاح:
- رجعت ألوان الواجهة إلى البيج والبني مثل التصميم السابق.
- لم يتم تغيير أشكال البطاقات أو ترتيب الصفحة.
- أزيلت تأثيرات Blur الثقيلة لتقليل التهنيگ على الهاتف.
- تم منع الضغط المتكرر السريع على إضافة للسلة.


تحديث جديد:
- خيار "تعديل الصفحة الرئيسية" داخل لوحة الإدارة.
- معاينة مباشرة لنفس صفحة البيع.
- التحكم بإظهار وإخفاء العناصر، النصوص، الأحجام، الأعمدة، الخلفية، وترتيب الأقسام.

تحديث هذه النسخة:
- زر واتساب يفتح المحادثة مع وصل الطلب مكتوباً وجاهزاً للإرسال.
- كميات المنتجات غير محدودة ويمكن إدخال 100 أو أكثر مباشرة.
